# 💍 Wedding Cloud - Dashboard de Mariage Sync

Une application web moderne et responsive pour organiser son mariage à deux. Les données sont synchronisées en temps réel entre PC et Mobile grâce à Firebase.

## ✨ Fonctionnalités

- 🔐 **Espace Privé** : Système d'inscription et de connexion pour chaque couple.
- 👥 **Multi-Utilisateurs** : Possibilité de basculer entre les deux profils du couple pour savoir qui a ajouté quoi.
- 📝 **Planning & Budget** : Ajoutez des tâches, définissez des catégories personnalisées (DJ, Fleurs, Traiteur...) et gérez les tarifs.
- 💶 **Calcul Automatique** : Le budget total est calculé et mis à jour en temps réel.
- ⏳ **Compte à rebours** : Affiche le nombre de jours restants avant le grand jour.
- 🎨 **Personnalisation** : Choisissez la couleur de fond de votre dashboard (synchronisé entre les appareils).
- 📱 **Responsive Design** : Interface optimisée pour PC, Tablettes et Smartphones.
